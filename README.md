# Trabalho da Aline

## Entrega: 01/07/26 (quarta-feira)

## O que será feito?

Vou desenvolver um algoritimo de **Regressão Linear** para **prevêr** a nota de um aluno baseado no tempo de estudo.

### Dados

Conjunto de notas de diversos alunos e suas horas estudadas.

### Entrada

Número de horas estudas de um aluno X.

### Saída

Qual é a nota mais provável que o aluno vai tirar?
